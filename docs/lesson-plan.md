# План объяснения на уроке

## Цель урока

Показать ученику, что веб-приложение состоит из нескольких ролей:

- браузер показывает интерфейс;
- JavaScript отправляет действия игрока на сервер;
- Python-сервер принимает URL-запросы и выполняет правила игры;
- текстовый файл хранит общий счет;
- домен и внешний веб-сервер помогают открыть приложение из интернета.

Не цель урока: подробно изучать CSS, Docker, HTTPS, DNS или промышленную безопасность. Их можно назвать как отдельные слои и не уходить в детали.

## Порядок занятия

1. Открыть `docs/scheme.md` и пройти путь запроса от пользователя до файла.
2. Открыть `static/index.html` и показать, что HTML - это заготовка экрана.
3. Открыть `static/game.js` и показать только крупные действия: старт, ответ, таймер, обновление счета.
4. Открыть `server.py` и найти блок `app.add_api_route(...)` внизу файла.
5. Разобрать таблицу URL из `README.md`.
6. Запустить игру локально командой `python3 server.py`.
7. Ввести имя, решить пару примеров, открыть `data/scores.txt`.
8. Объяснить, что на домене работает тот же проект, только перед ним стоит Caddy.

## Главная часть для ученика

Самое важное место в коде:

```python
app.add_api_route("/", page_index, methods=["GET", "HEAD"])
app.add_api_route("/api/start", api_start, methods=["POST"])
app.add_api_route("/api/answer", api_answer, methods=["POST"])
```

Это показывает идею веб-сервера без декораторов:

```text
пришел URL -> FastAPI вызвал нужную функцию -> функция вернула ответ
```

## Что можно дать как маленькое задание

- Добавить деление без остатка.
- Увеличить время игры до 90 секунд через `GAME_SECONDS`.
- Сделать за правильный ответ `+10` очков вместо `+1`.
- Добавить отдельную строку "последний правильный ответ".
- Поменять формат файла счета и обсудить, почему это опасно делать неаккуратно.

## Где остановиться

Если ученик еще не знает классов, не надо разбирать внутренности FastAPI и Uvicorn. Достаточно сказать:

```text
FastAPI - готовый инструмент, который связывает URL с функциями.
Uvicorn - программа, которая запускает FastAPI-сервер.
```
